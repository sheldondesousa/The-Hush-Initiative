# HUM Sound Therapy

A web application for hum sound therapy with a beautiful login interface.

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/sheldondesousa/hum-sound-therapy.git
cd hum-sound-therapy
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser to `http://localhost:5173`

## Features

- Modern, responsive login screen
- Social authentication (Apple, Google)
- Clean UI based on Figma design
- Mobile-first design
